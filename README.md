# Person Profiler

**Person profiling engine with 7-axis analysis.**

Web research-based: Career DNA, Decision Patterns, Vision, Reputation, Personality, Network, Current Context → DNA traits → strengths/weaknesses → 30-second elevator pitch.

### Example Prompts

```
"Profile the CEO of [company] for a meeting" → 7-axis research→DNA→elevator pitch
"Research [person] for investor prep" → adds Network axis for business context
```
