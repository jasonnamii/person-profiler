# P3 프로파일링 상세 규칙

## DNA 도출 규칙

- 전 축 조사 결과에서 **반복 패턴** 추출 → DNA 5~7대 명명
- 각 DNA: 명칭(한 단어~구) + 증거(구체적 사례 1+) + 강도(⭐1~5)
- 증거 없는 DNA 금지. 추론만으로 도출 시 "추정" 태그 필수

## 강점·약점 판정

- 전 축 분석 결과를 수렴하여 강점 3~5개, 약점 3~5개 도출
- 각 항목: 1줄 판정 + 근거 축 번호 명시
- 강점과 약점이 동전의 양면인 경우 명시 (예: "추진력 ↔ 독단")

## 의사결정 패턴

- 핵심 의사결정 사례 3개 이상에서 공통 루프 추출
- 표현: 텍스트 플로우 or 테이블 (Mermaid는 형 요청시만)
