# §3 분석 중간 산출물 구조

P1~P4 파이프라인의 **분석 결과물**. 이 단계는 데이터 구조이며 최종 리포트가 아니다.

```
§0 이미지 앵커
   - 인물 대표 이미지 (웹서치 수집 URL, ![alt](url) 형식)
   - 이미지 미확보시 생략 (빈 칸 금지)

§1 엘베피치 (30초 버전)
   - Layer 1→2→3 수렴 결과
   - 전체 리포트의 스파인

§2 기본 프로필
   - 이름 | 직함·소속 | 학력 | 핵심 경력(5줄 이내)

§3 DNA 프로파일링
   - 5~7대 DNA, 각각: 명칭 | 증거 | 강도(⭐)

§4 의사결정 패턴
   - 핵심 사례 3+ → 공통 루프 도출
   - 리스크 수용도: 높음/중간/낮음 (영역별)

§5 강점·약점 판정
   - 강점 3~5개 | 약점 3~5개
   - 동전의 양면 관계 표기

§6 현재 컨텍스트 & 방향성
   - 최근 동향 (시간순 3~5개) + 향후 예상

§7 설득 포인트 / 접근 전략
   - High Fit 3개 | Low Fit 3개 | 추천 접근법

§8 리스크 & 맹점
   - 분석 한계 (소스 편향·정보 공백)
   - 인물 자체 리스크
```

**CRITICAL**: 이 중간 산출물은 **내부 분석용**이다. 형에게 전달하는 최종 리포트는 §3.5에서 변환한다.
